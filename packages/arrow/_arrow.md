# Arrow

## 简介

R 语言的 **arrow 包**让 R 用户可以使用 Apache Arrow C++ 库的众多功能。arrow 包的目标是为 dplyr 提供 Arrow C++ 后端，并通过大家熟悉的**基础 R 函数、tidyverse 函数或 R6 类**来访问 Arrow C++ 库。

关于 **Apache Arrow** 的更多信息，请查看 [Arrow 主项目](https://arrow.apache.org/)的文档。Arrow 项目为各类数据分析任务提供相应功能，用于**高速存储、处理和传输数据**。
