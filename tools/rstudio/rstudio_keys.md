# RStudio 快捷键

## 面板跳转

| 快捷键 | 窗口              |
| ------ | ----------------- |
| Ctrl+1 | source editor     |
| Ctrl+2 | console           |
| Ctrl+3 | workspace browser |
| Crtl+4 | history           |
| Crtl+5 | file browser      |
| Crtl+6 | plots             |
| Crtl+7 | packages          |
| Crtl+8 | help              |
| Crtl+9 | git               |
